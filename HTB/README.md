# Hack The Box
## Pengenalan Hack The Box
Hack The Box adalah platform yang menyediakan mesin yang memiliki celah keamanan. Untuk dapat menggunakan platform ini, kalian dapat mendaftar secara gratis. Namun untuk mendaftarnya kalian harus menyelesaikan challenge. Dalam platform ini kalian dapat melaukan subscribe untuk mendapatkan layanan penuh.
